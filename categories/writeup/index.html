Handles in 2016

Sep 23, 2016 Identity Usernames Handle Nickname IRC

For years I used the name “Daemons Advocate”. I don’t know why, but at some stage I stopped liking it. This got tricky, anytime I get to know a name I start to dislike it. Picking a handle is too hard. And everything I came up with is taken by some other schmo on the internets because if you’re looking for a good short name in 2016, you’re way late to the party.
